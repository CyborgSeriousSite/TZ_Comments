Техническое задание (Реализация сообщений):
Mysql база носит наименование - tzchat. Включает в себя табицу usermessages.

Состав usermessages:
* id - Тип int, autoincrement.
* fio - Тип tinytext
* email - Тип tinytext
* content - Тип text
* publicationtime - Тип timestamp. Автоматически назначает время, при создании записи (CURRENT_TIME)

Изначально data располагалась по пути C:\Server\
В этом пути располагалась папка bin в которой был apache (httpd-2.4.48), php 8.0.7, mysql-8.0.25-winx64
Внутри data файлы сайта.

-- Состав ТЗ --
Обязательные требования: 
1. Сообщения должны сохраняться в бд PostgreSQL, MySQL или MariaDB. 
2. Форма должна содержать следующие поля: 
ФИО, email, текст сообщения и кнопку отправки. 
Дизайн придумайте сами. 
3. Поля должны проходить обязательную валидацию на стороне сервера. 
4. Должны быть соблюдены меры для защиы от SQL инъекций и XSS. 
5. На странице должен быть осуществлен вывод уже отправленных ранее сообщений. 
6. Работа с бд должна происходить с использованием PDO. 
7. Все перечисленное должно быть выполнено без использования фреймворков. 
8. Верстка формы должна быть выполнена с соблюдением стандартов HTML5 и CSS3. 
9. Список сообщений должен обновляться с использованием технологии ajax(можно и fetch).
10. Форма должна отправляться с использованием технологии ajax(можно и fetch). 
11. JavaScript должен быть написан по стандарту ES6+(можно использовать классы и promises).
12. Весь проект должен быть выполнен с использованием CVS GIT и содержать несколько коммитов.
13. php код должен быть обьектным и с использованием MVC шаблона проектирования.

Требования к оформлению php кода:
1. без использования фреймворков и библиотек типа jQuery.
2. только ООП.
3. должен быть использован MVC шаблон проектирования.
4. исходный код должен быть оформлен по PSR-1, PSR-2.
5. Код должен легко читаться.

Требования к оформлению JavaScript коду:
1. С использрванием ECMAScript 6.
2. Код должен легко читаться.

Требования к оформлению HTML:
1. Разметка должна хорошо читаься.
2. без шаблонизаторов.
3. по стандарту HTML5.

Требования к оформлению CSS:
1. Код должен легко читаться.
2. По стандарту CSS3.

Какие знания потребуются для выполнения задания: 
1. Знание скриптовых языков программирования PHP, JavaScript(ES6+). 
2. Основы языков разметки HTML5, CSS3. 
3. Язык запросов SQL.
