#  Техническое задание (Реализация сообщений)
За основу взят denwer, представляющий набор дистрибутивов и программную оболочку, предназначенную для создания и отладки сайтов на локальном ПК под управлением ОС Windows.
Имеет встроенный apache сервер, PHP 5.3.13, MySQL 5.1, PostgreSQL 8.4 и т.д.
В конфигурации поставлен диск Z, который создается при запуске сервера.
Уже имеет настроенную пустую базу данных. Носит название tzchat. Включает в себя табицу usermessages.

Состав usermessages:
* id - Тип int, autoincrement.
* fio - Тип tinytext
* email - Тип tinytext
* content - Тип text
* publicationtime - Тип timestamp. Автоматически назначает время, при создании записи (CURRENT_TIME)

## Как запустить
Запуск сервера находится по пути: корневая_папка_проекта\denwer\Run.exe
